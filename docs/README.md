# Documentação do SOS Moçambique

## O que é o SOS Moçambique

O `SOS Moçambique` é actualmente um protótipo frontend-first para resposta humanitária, construído com `React + Vite`, focado em pedidos de ajuda, relatos de resgate, reencontro familiar, mapa de zonas seguras e administração local de dados simulados.

## Estado actual do projecto

- protótipo funcional de frontend;
- persistência simulada com `localStorage`;
- sem backend real;
- sem base de dados real;
- sem autenticação real;
- com algumas interacções simuladas para demonstrar o fluxo do produto.
