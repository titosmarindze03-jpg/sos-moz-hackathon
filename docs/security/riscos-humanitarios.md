# Riscos Humanitários

## Riscos específicos

- falsa sensação de socorro real;
- exposição de dados pessoais;
- localização de pessoas vulneráveis;
- necessidade de validação institucional;
- necessidade de integração com entidades oficiais antes de uso real.

## Regra crítica

Este protótipo não deve ser usado como canal oficial de emergência até existir validação técnica, legal e institucional.
