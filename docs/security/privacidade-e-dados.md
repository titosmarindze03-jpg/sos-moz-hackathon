# Privacidade e Dados

## Estado actual

Os dados são guardados no navegador do utilizador através de `localStorage`.

## Riscos

- exposição local dos dados;
- perda de dados ao limpar o navegador;
- ausência de controlo institucional;
- ausência de política de retenção;
- ausência de consentimento formal documentado.

## Observação

Como o projecto lida com contexto humanitário, qualquer evolução real deve tratar dados pessoais com protecção reforçada.
