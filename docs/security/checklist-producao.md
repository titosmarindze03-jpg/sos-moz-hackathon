# Checklist Produção

## Antes de considerar produção

- backend real implementado;
- base de dados real implementada;
- autenticação real implementada;
- autorização por perfis implementada;
- auditoria implementada;
- protecção de dados pessoais definida;
- integração institucional validada;
- geolocalização real validada;
- canais reais de contacto validados;
- segurança revista.

## Estado actual

Nenhum destes pontos está concluído de forma suficiente para produção.
