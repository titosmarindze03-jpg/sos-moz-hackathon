# Estilos e UI

## Características da UI

- foco mobile-first;
- interface baseada em cartões;
- uso forte de cores de alerta e emergência;
- navegação por topo, menu lateral e barra inferior;
- componentes visuais orientados ao contexto humanitário.

## Estilo actual

- fonte principal remota;
- ícones remotos;
- estilos utilitários;
- aparência de aplicação móvel.

## Regra

Qualquer evolução da UX deve preservar:

- português como língua principal;
- leitura rápida;
- distinção clara entre fluxos de cidadão e admin;
- foco em uso móvel.
