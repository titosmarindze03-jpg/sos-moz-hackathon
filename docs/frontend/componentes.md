# Componentes

## Componentes principais

- `App.tsx`
- `TopAppBar.tsx`
- `BottomNavBar.tsx`
- `DashboardView.tsx`
- `RequestHelpView.tsx`
- `ReportIsolatedView.tsx`
- `FamilyView.tsx`
- `FullMapView.tsx`
- `AdminView.tsx`

## Resumo de responsabilidade

- `App.tsx`: composição principal e controlo da vista activa;
- `TopAppBar.tsx`: topo com menu e atalho SOS;
- `BottomNavBar.tsx`: navegação inferior;
- `DashboardView.tsx`: entrada principal;
- `RequestHelpView.tsx`: pedidos de ajuda;
- `ReportIsolatedView.tsx`: relato de pessoas isoladas;
- `FamilyView.tsx`: reencontro familiar;
- `FullMapView.tsx`: mapa visual;
- `AdminView.tsx`: gestão administrativa local.
