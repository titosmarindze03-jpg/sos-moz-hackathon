# Modelos

## Modelos actuais

- `Person`
- `Alert`
- `SafeZone`
- `HelpRequest`
- `RescueReport`

## Papel

Estes modelos definem os contratos tipados usados pelo frontend actual para representar dados do protótipo.

## Estado

São modelos de frontend. Não constituem, por si só, um contrato oficial de API externa.
