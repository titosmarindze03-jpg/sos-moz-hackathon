# Leia Primeiro

O `SOS Moçambique` ainda é um protótipo.

## Avisos críticos

- o projecto ainda é protótipo;
- os dados actuais são simulados e guardados no `localStorage`;
- não existe backend real implementado;
- não existe autenticação real;
- não existe GPS real integrado;
- não existe SMS real integrado;
- não existe IA real integrada;
- não deve ser usado como sistema real de emergência ainda.

## O que este protótipo faz hoje

- demonstra fluxos de ajuda humanitária no frontend;
- permite registos locais de pedidos, alertas, pessoas e zonas;
- simula alguns comportamentos operacionais para fins de demonstração.

## O que este protótipo não faz

- não envia pedidos para entidades reais;
- não partilha dados entre dispositivos;
- não protege acesso administrativo de forma robusta;
- não valida informação por backend;
- não garante fiabilidade institucional.

## Regra documental

Toda a documentação desta pasta deve distinguir claramente entre:

- funcionalidades implementadas;
- funcionalidades simuladas;
- funcionalidades planeadas.
