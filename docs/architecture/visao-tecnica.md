# Visão Técnica

O projecto actual segue uma abordagem frontend-first:

- `React` para interface;
- `TypeScript` para tipagem;
- `Vite` para desenvolvimento e build;
- `localStorage` para persistência simulada;
- serviços locais para encapsular leitura e escrita.

## Estado técnico actual

- aplicação monolítica de frontend;
- sem backend integrado;
- sem API externa activa;
- sem base de dados externa;
- sem autenticação.
