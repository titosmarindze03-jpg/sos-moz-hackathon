# Dados e Persistência

## Persistência actual

A persistência actual é feita no `localStorage` do navegador.

## Natureza dos dados

- local;
- não sincronizada;
- sem backend;
- sem histórico auditável robusto;
- adequada apenas para protótipo.

## Entidades observadas

- pessoas;
- alertas;
- pedidos de ajuda;
- relatos de resgate;
- zonas seguras.
