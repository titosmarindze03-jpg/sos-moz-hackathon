# Público-Alvo

## Utilizadores principais

- cidadãos afectados por eventos críticos;
- voluntários ou operadores de apoio local;
- administradores do sistema em contexto de protótipo.

## Distinção funcional actual

- cidadão: utiliza fluxos de ajuda, família, mapa e alertas;
- admin: utiliza o painel interno para gerir dados locais simulados.

## Observação

Esta distinção existe ao nível da interface e do fluxo, mas não existe controlo real de autenticação ou autorização.
