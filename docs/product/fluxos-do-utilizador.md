# Fluxos do Utilizador

## Cidadão

1. abre o painel principal;
2. consulta alertas ou pontos seguros;
3. pede ajuda;
4. reporta pessoa isolada;
5. procura familiares;
6. consulta mapa.

## Administrador

1. abre o painel administrador;
2. gere pessoas;
3. gere pedidos de ajuda;
4. gere relatos de resgate;
5. gere alertas;
6. gere zonas seguras e de risco.

## Nota

Os fluxos actuais são locais ao frontend e não representam operação em rede real.
