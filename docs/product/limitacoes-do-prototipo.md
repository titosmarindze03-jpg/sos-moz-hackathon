# Limitações do Protótipo

- não existe backend real;
- não existe base de dados real;
- não existe autenticação real;
- não existe partilha multiutilizador;
- os dados podem perder-se ao limpar o navegador;
- várias interacções são simuladas;
- não há garantia de entrega de informação a entidades reais;
- o sistema não está pronto para produção.
