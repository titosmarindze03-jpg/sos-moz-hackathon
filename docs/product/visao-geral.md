# Visão Geral

O `SOS Moçambique` é um protótipo digital concebido para apoiar cenários de resposta humanitária, com foco em:

- pedidos de ajuda;
- relatos de pessoas isoladas;
- reencontro familiar;
- alertas informativos;
- zonas seguras e de risco;
- gestão administrativa local de informação.

Actualmente, o sistema existe como frontend demonstrativo e não como plataforma operacional completa.
